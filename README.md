# 🎮 遊戲清單網頁應用程式

這是一個基於 **React** 的遊戲清單網頁應用程式，用於展示和管理遊戲清單。使用者可以瀏覽遊戲資訊。

---

## 🚀 功能特色

### 🕹️ 遊戲清單
- 展示遊戲清單，包含以下資訊：
  - **遊戲名稱**
  - **遊戲類型**
  - **遊戲標籤**
  - **遊戲網頁**
- 每個遊戲以 `<article>` 形式呈現，簡潔直觀。

---

## 🛠️ 使用技術

- **React**：負責構建元件化的使用者介面。
- **CSS/SCSS**：用於網頁樣式及響應式設計。
